# Robotic Guide Dog Site

Standalone project site for the robotic guide dog work.

## Local preview

From `/Users/shaojun/Desktop/shaojun-site/robotic-guide-dog-site`:

```bash
python3 -m http.server 4321
```

Then open `http://localhost:4321`.

## Structure

- `index.html`: single-page project site
- `styles.css`: visual system and layout
- `script.js`: reveal-on-scroll motion
- `assets/images/`: local project media copied from the main site
